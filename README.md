## 1. Что такое интеграл

Интеграл — это одна из двух фундаментальных операций математического анализа (вторая — производная).

Геометрически **определённый интеграл** `∫[a,b] f(x) dx` — это площадь фигуры, ограниченной графиком функции `f(x)`, осью `x` и вертикальными прямыми `x = a` и `x = b`.

```
  f(x)
   │     ╭───╮
   │    ╭╯   ╰╮
   │   ╭╯     ╰╮
   │───╯         ╰───
   └──────────────── x
        a       b
        └───────┘
         площадь = ∫[a,b] f(x) dx
```

**Неопределённый интеграл** `∫ f(x) dx` — это семейство всех первообразных функции `f(x)`, то есть всех функций `F(x)` таких, что `F'(x) = f(x)`.

---

## 2. Неопределённый интеграл

### Определение

Функция `F(x)` называется **первообразной** функции `f(x)` на интервале, если:

```
F'(x) = f(x)  для всех x из области определения
```

Неопределённый интеграл записывается как:

```
∫ f(x) dx = F(x) + C
```

где `C` — произвольная константа интегрирования. Она появляется потому, что производная любой константы равна нулю, и поэтому первообразная определена с точностью до константы.

### Пример

```
f(x) = x²

F(x) = x³/3  →  F'(x) = x²  ✓

∫ x² dx = x³/3 + C
```

### Линейность интеграла

Интеграл — линейный оператор:

```
∫ [α·f(x) + β·g(x)] dx = α·∫f(x)dx + β·∫g(x)dx
```

Это позволяет разбивать сложные выражения на сумму простых.

## 3. Определённый интеграл и формула Ньютона–Лейбница

### Определение через суммы Римана

Определённый интеграл определяется как предел интегральных сумм:

```
∫[a,b] f(x) dx = lim(n→∞) Σ f(xᵢ) · Δx
                             i=1..n
```

где `Δx = (b−a)/n`, а `xᵢ = a + i·Δx` — точки разбиения отрезка `[a, b]`.

### Формула Ньютона–Лейбница

Это главная теорема математического анализа. Она связывает неопределённый и определённый интеграл:

```
∫[a,b] f(x) dx = F(b) − F(a)
```

где `F(x)` — любая первообразная функции `f(x)`.

### Как это реализовано в коде

```python
# core/solver.py — функция solve_definite()

antiderivative    = sp.simplify(sp.integrate(expression, variable_x))
definite_result   = antiderivative.subs(variable_x, upper_bound) \
                  - antiderivative.subs(variable_x, lower_bound)
simplified_result = sp.simplify(definite_result)
```

1. Сначала находится первообразная `F(x)` символьно через `sp.integrate`
2. Затем вычисляется `F(b) − F(a)` подстановкой через `.subs()`
3. Результат упрощается через `sp.simplify()`

### Пример

```
∫[0,1] x² dx

F(x) = x³/3

F(1) − F(0) = 1/3 − 0 = 1/3 ≈ 0.333333
```

## 4. Как приложение определяет метод

Перед вычислением `solver.py` анализирует структуру выражения и выбирает метод. Порядок проверок важен — каждая следующая проверяется только если предыдущая вернула `False`:

```
Выражение f(x)
     │
     ├─ является дробью полиномов?          → Partial Fractions
     │
     ├─ содержит log/arctan × полином?      → Integration by Parts
     │
     ├─ содержит sin/cos/tan/...?           → Trigonometric Integration
     │
     ├─ содержит дробную степень (корень)?  → Irrational / Root Substitution
     │
     ├─ содержит f(g(x)) где g(x) ≠ x?     → U-Substitution
     │
     └─ ничего из вышеперечисленного        → Direct Integration
```

### Проверка рациональной функции (`_is_rational_function`)

```python
numerator, denominator = sp.fraction(sp.together(expression))
# sp.together() приводит к общему знаменателю
# sp.fraction() разделяет на числитель и знаменатель
# затем проверяем, что оба — полиномы от x
sp.Poly(numerator, variable_x)
sp.Poly(denominator, variable_x)
```

Пример: `1/(x²−1)` → числитель `1`, знаменатель `x²−1` — оба полиномы ✓

### Проверка дробной степени (`_contains_irrational_power`)

```python
power.exp.is_Rational and power.exp.p % power.exp.q != 0
```

Степень `p/q` является дробной (не целой), если `p` не делится на `q`.
Пример: `sqrt(x) = x^(1/2)` → `p=1, q=2, 1 % 2 = 1 ≠ 0` ✓

## 5. Метод 1 — Прямое интегрирование

Применяется когда выражение совпадает со стандартной табличной формулой.

### Основные табличные интегралы

| Функция f(x) | Первообразная F(x) | Условие |
|---|---|---|
| `xⁿ` | `xⁿ⁺¹ / (n+1)` | `n ≠ −1` |
| `1/x` | `ln\|x\|` | `x ≠ 0` |
| `eˣ` | `eˣ` | |
| `aˣ` | `aˣ / ln(a)` | `a > 0, a ≠ 1` |
| `sin(x)` | `−cos(x)` | |
| `cos(x)` | `sin(x)` | |
| `1/(1+x²)` | `arctan(x)` | |
| `1/√(1−x²)` | `arcsin(x)` | |
| `sinh(x)` | `cosh(x)` | |
| `cosh(x)` | `sinh(x)` | |

### Пример

```
∫ x⁵ dx = x⁶/6 + C     (n=5, применяем формулу xⁿ)

∫ eˣ dx = eˣ + C        (таблица)

∫ cos(x) dx = sin(x) + C
```

## 6. Метод 2 — Метод подстановки (замена переменной)

### Теорема

Если `g(x)` — дифференцируемая функция и `u = g(x)`, то:

```
∫ f(g(x)) · g'(x) dx = ∫ f(u) du
```

### Алгоритм

```
1. Найти внутреннюю функцию g(x) в составном выражении
2. Ввести замену:  u = g(x)
3. Вычислить:      du = g'(x) dx  →  dx = du / g'(x)
4. Переписать интеграл целиком через u
5. Вычислить ∫ f(u) du  (обычно это табличный интеграл)
6. Вернуться к x:  подставить u = g(x)
```

### Пример

```
∫ sin(x²) · 2x dx

Замена:  u = x²
         du = 2x dx

∫ sin(u) du = −cos(u) + C = −cos(x²) + C
```

### Как определяется в коде

```python
for func_atom in expression.atoms(sp.Function):
    inner_arg = func_atom.args[0]
    if inner_arg != variable_x and not inner_arg.is_Number:
        return "U-Substitution"
```

Если аргумент функции — не просто `x` и не константа, значит это составная функция `f(g(x))`.

## 7. Метод 3 — Интегрирование по частям

### Формула

Выводится из правила произведения для производных:

```
(u·v)' = u'·v + u·v'
```

Интегрируя обе части:

```
∫ u·v' dx = u·v − ∫ v·u' dx
```

В стандартной записи через дифференциалы:

```
∫ u dv = u·v − ∫ v du
```

### Правило LIATE для выбора u

Выбирай `u` в следующем приоритете (первое в списке — лучший кандидат):

```
L — Logarithms     (log, ln)
I — Inverse trig   (arctan, arcsin, arccos)
A — Algebraic      (xⁿ, полиномы)
T — Trigonometric  (sin, cos)
E — Exponential    (eˣ)
```

`dv` — всё остальное.

### Пример

```
∫ x · ln(x) dx

По LIATE:  u = ln(x)  (L стоит выше A)
           dv = x dx

du = (1/x) dx
v  = x²/2

∫ x·ln(x) dx = ln(x)·(x²/2) − ∫ (x²/2)·(1/x) dx
             = (x²/2)·ln(x) − ∫ x/2 dx
             = (x²/2)·ln(x) − x²/4 + C
```

### Как реализовано в коде

```python
# core/solver.py — _steps_by_parts()

log_arc_factors   = [f for f in all_factors if f.atoms(sp.log, sp.atan, sp.asin, sp.acos)]
remaining_factors = [f for f in all_factors if f not in log_arc_factors]

u_part  = log_arc_factors[0]          # логарифм или арктангенс → u
dv_part = sp.Mul(*remaining_factors)  # остаток → dv

du_part = sp.diff(u_part, variable_x)
v_part  = sp.integrate(dv_part, variable_x)
```

## 8. Метод 4 — Разложение на простые дроби

Применяется для рациональных функций вида `P(x)/Q(x)`, где `P` и `Q` — полиномы.

### Идея

Любую правильную рациональную дробь (степень числителя < степени знаменателя) можно разложить в сумму простых дробей, каждую из которых легко интегрировать.

### Типы простых дробей и их интегралы

```
A/(x − a)           →  A · ln|x − a|

A/(x − a)²          →  −A/(x − a)

(Ax + B)/(x² + px + q)   →  через arctg после выделения полного квадрата
```

### Алгоритм

```
1. Если степень P ≥ степени Q — выполнить деление полиномов
2. Разложить знаменатель Q(x) на множители над ℝ
3. Записать разложение на простые дроби с неизвестными A, B, C, ...
4. Найти коэффициенты (метод неопределённых коэффициентов или подстановка корней)
5. Интегрировать каждое слагаемое по таблице
```

### Пример

```
∫ 1/(x²−1) dx = ∫ 1/((x−1)(x+1)) dx

Разложение:  1/((x−1)(x+1)) = A/(x−1) + B/(x+1)

Умножаем на (x−1)(x+1):
  1 = A(x+1) + B(x−1)

x = 1:   1 = 2A  →  A = 1/2
x = −1:  1 = −2B →  B = −1/2

∫ 1/(x²−1) dx = (1/2)·ln|x−1| − (1/2)·ln|x+1| + C
              = (1/2)·ln|(x−1)/(x+1)| + C
```

### Как реализовано в коде

```python
# core/solver.py — _steps_partial_fractions()

decomposed_expr  = sp.apart(expression, variable_x)   # SymPy разлагает на дроби
individual_terms = sp.Add.make_args(decomposed_expr)   # разбиваем сумму на слагаемые
integrated_terms = [sp.integrate(term, variable_x) for term in individual_terms]
```

`sp.apart()` реализует алгоритм разложения на простые дроби через расширенный алгоритм Евклида для полиномов.

## 9. Метод 5 — Тригонометрические интегралы

Применяется когда подынтегральное выражение содержит `sin`, `cos`, `tan`, `cot`, `sec`, `csc`.

### Основные тождества для понижения степени

```
sin²(x) = (1 − cos(2x)) / 2
cos²(x) = (1 + cos(2x)) / 2
sin(x)·cos(x) = sin(2x) / 2
```

### Случай ∫ sinⁿ(x) cosᵐ(x) dx

**Если n нечётное** — замена `u = cos(x)`:
```
∫ sin³(x) dx = ∫ sin²(x)·sin(x) dx = ∫ (1−cos²x)·sin(x) dx
u = cos(x), du = −sin(x) dx
= −∫ (1−u²) du = −u + u³/3 + C = −cos(x) + cos³(x)/3 + C
```

**Если m нечётное** — замена `u = sin(x)`.

**Если оба чётные** — применяем формулы двойного угла.

### Универсальная тригонометрическая подстановка

Замена `t = tan(x/2)` сводит любой тригонометрический интеграл к рациональному:

```
sin(x) = 2t/(1+t²)
cos(x) = (1−t²)/(1+t²)
dx     = 2dt/(1+t²)
```

### Стандартные результаты

```
∫ sin(x) dx  = −cos(x) + C
∫ cos(x) dx  =  sin(x) + C
∫ tan(x) dx  = −ln|cos(x)| + C
∫ cot(x) dx  =  ln|sin(x)| + C
∫ sec²(x) dx =  tan(x) + C
∫ csc²(x) dx = −cot(x) + C
```

## 10. Метод 6 — Иррациональные выражения (корни)

Применяется когда подынтегральное выражение содержит `√(...)` или дробную степень.

### Определение в коде

```python
# core/solver.py — _contains_irrational_power()

power.exp.is_Rational and power.exp.p % power.exp.q != 0
```

Степень `p/q` является иррациональной (в смысле «не целой»), если остаток от деления `p` на `q` не равен нулю. Например: `x^(1/2)` → `1 % 2 = 1 ≠ 0`.

### Алгебраическая подстановка

Для `√(ax + b)` вводим `t = √(ax + b)`:

```
∫ x·√(x+1) dx

t = √(x+1)  →  t² = x+1  →  x = t²−1,  dx = 2t dt

∫ (t²−1)·t · 2t dt = 2∫ (t⁴ − t²) dt
= 2·(t⁵/5 − t³/3) + C
= 2(x+1)^(5/2)/5 − 2(x+1)^(3/2)/3 + C
```

### Тригонометрическая подстановка

Для выражений вида `√(a²−x²)`, `√(a²+x²)`, `√(x²−a²)`:

| Выражение | Подстановка | Результат |
|---|---|---|
| `√(a²−x²)` | `x = a·sin(t)` | `√(a²−x²) = a·cos(t)` |
| `√(a²+x²)` | `x = a·tan(t)` | `√(a²+x²) = a/cos(t)` |
| `√(x²−a²)` | `x = a/cos(t)` | `√(x²−a²) = a·tan(t)` |

### Пример

```
∫ 1/√(1−x²) dx

x = sin(t)  →  dx = cos(t) dt
√(1−x²) = √(1−sin²t) = cos(t)

∫ cos(t)/cos(t) dt = ∫ dt = t + C = arcsin(x) + C
```

## 11. Как SymPy вычисляет интеграл внутри

SymPy — это система компьютерной алгебры (CAS). Она работает с математическими выражениями **символьно**, а не численно.

### Внутренний алгоритм `sp.integrate()`

SymPy применяет алгоритмы в следующем порядке:

#### 1. Алгоритм Риша (Risch algorithm)
Полный алгоритм для элементарных функций. Определяет, существует ли первообразная в классе элементарных функций, и если да — находит её. Работает для выражений из `exp`, `log`, алгебраических функций.

#### 2. Эвристика Манна (Meijer G-functions)
Для специальных функций, которые не берутся в элементарных функциях (например, `erf`, `Si`, `Ci`).

#### 3. Таблица паттернов
Для простых выражений SymPy сначала проверяет встроенную таблицу паттернов — это быстрее полного алгоритма.

### Парсинг входной строки

```python
# core/solver.py

expression = sp.sympify(expression_str, locals={"x": variable_x})
```

`sp.sympify()` преобразует строку `"x**2 + sin(x)"` в дерево символьных выражений:

```
Add
├── Pow
│   ├── Symbol('x')
│   └── Integer(2)
└── sin
    └── Symbol('x')
```

Именно это дерево анализируется методами `.atoms()`, `.args`, `sp.fraction()` и т.д.

### Упрощение результата

```python
antiderivative = sp.simplify(sp.integrate(expression, variable_x))
```

`sp.simplify()` применяет набор эвристик: раскрытие скобок, тригонометрические тождества, логарифмические свёртки, и выбирает наиболее компактную форму.

### Численное значение для определённого интеграла

```python
numeric_value = str(float(simplified_result.evalf(6)))
```

`.evalf(6)` вычисляет символьное выражение с точностью до 6 значащих цифр, используя произвольную точность арифметику (библиотека `mpmath` внутри SymPy).
