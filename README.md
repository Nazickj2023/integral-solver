# Integral Solver

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Nazickj2023/integral-solver.git
   ```
2. Navigate to the project directory:
   ```bash
   cd integral-solver
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Features
- Compute definite and indefinite integrals.
- Supports multiple variables.
- User-friendly interface.

## Usage
1. Start the application:
   ```bash
   npm start
   ```
2. Enter the integral expression in the input field.
3. Click on the calculate button.
4. View the result on the output display.
